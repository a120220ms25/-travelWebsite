## 內容
使用 Bootstrap 為開發工具與相關套件，結合交通部觀光資料開放平台、中央氣象局開放資料平台 Open Data 做為 API 串接數據進行即時資訊，旅遊網站設計。

##
1. 編寫 Ajax 取得 JSON 數據。
2. 使用JQuery操作DOM元素，控制頁面顯示。
3. 整合Google Analytics套件，利用網站流量統計服務，進行使用者分析。
4. 使用 DataTables.JS 套件進行視覺美化。
